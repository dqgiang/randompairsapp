# Random Pairs App

## Prerequisites

- .NET Core 2.x with CLI tool installed
- A web browser
- Port 5000 at localhost is available

## How to run

### Backend service

- Open the root folder (`RandomPairsApp`) in a terminal (or command prompt)
- Run `dotnet build`
- Run `dotnet run`

The above run commands should succeed.
